**N3MO : une couche d’intelligence de code déterministe sans LLM**
N3MO est une couche d’intelligence de code déterministe qui transforme le code source en un graphe de connaissances interrogeable via une base de données relationnelle, sans appel aux LLM à l’indexation. Il permet d’analyser les dépendances, les appels de fonctions et les impacts de modifications avec une précision de 50ms, même sur des codebases complexes. Compatible avec 27 langages via Tree-sitter, il offre une intégration native avec des agents IA et des outils comme Cursor ou Claude Desktop. **Ved : un agent IA vocal local et hors ligne**
Ved est un agent IA local fonctionnant entièrement hors ligne avec des modèles locaux (12 Go VRAM requis). Il intègre un orchestrateur LangGraph avec trois modes (chat standard, turbo/coder avec outils, hibernation), une interface Tkinter, un serveur FastAPI, et une pipeline vocale offline. Il supporte le RAG par thread, l’exécution de scripts Python, et offre une option cloud via OpenRouter. Les données de télémétrie sont locales et désactivables.
